# Product Brief: KM Bot v1

## Purpose

KM Bot v1 is a quick knowledge management assistant that helps users ask questions over a curated set of internal source materials. The prototype should prioritize trust, source grounding, and fast iteration over a polished interface.

## Initial Users

The initial user is an operator, founder, or chief-of-staff-style user who needs fast answers from scattered internal documents, research files, notes, and spreadsheets.

## First Milestone

The first milestone is a reliable Q&A loop over two or three approved files. The bot should return concise answers, cite source material, and flag when it does not have enough context.

## Non-Goals for v1

This version does not need a full Slack integration, complex permissions model, vector database, or multi-user admin console. Those can be added after the core knowledge-answering loop is proven.
